# Game Review Site

Projeto de Landing page de um site de review, ofertas, guias e atualizações de jogos, para aprendizado de novas técnicas e habilidades utilizadas diretamente no mercado.

Link do site: https://matheus-veloso-lino.github.io/game-review-site/

## Técnologia usada 
- CSS
- HTML

## Futuras atualizações

Atualmente possui apenas o hub do site, futuras atualizações será: 
- Página de cadastro/login para usuários.
- Pagina de reviews para reviews feitas diretamente por players cadastrados no site.
- Página de atualização escrita por mods/admins do site usando informação de sites como IGN, Steam e etc...
- Página de ofertas ligada diretamente á sites como SteamDB e IsThereAnyDeal, para ter ofertas em tempo real.
- Página de Guias escritos por mods, admins e usuários cadastrados no site.
